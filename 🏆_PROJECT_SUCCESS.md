## 🚀 使用方式

### 快速开始

```bash
# 1. 安装依赖
cd D:\WorkBench\ldesign\libraries\map
pnpm install

# 2. 启动示例
cd examples/vanilla
pnpm exec vite --port 3001 --host

# 3. 打开浏览器
# 访问: http://localhost:3001
```

### 开发命令

```bash
# 构建所有包
pnpm build

# 开发模式
pnpm dev

# 运行测试
pnpm test

# 类型检查
pnpm typecheck
```
